Key Features of the System
1) User Management: Admins can create, update, and delete user accounts ( HR staff, managers).
2) Employee Management: Add new employees, update existing records, and remove employees.
3) Role-based Access Control: Only authorized users can access certain functionalities
4) Search & Filter: Admins can search for employees based on different criteria (e.g., by department or designation)
5) Dashboard: Provides a comprehensive view of employee stats, performance, etc.
6) Security: Spring Security ensures that sensitive actions like modifying employee data or deleting records are protected.
